Technology

Powered by Zero Knowledge Proofs

Zero-knowledge Proofs (ZKPs) is a powerful cryptographic construction that protect your privacy on chain-purely from the first principle. Unlike some other technologies (i.e. Trusted Execution Environment, which need to trust the hardware vendor), ZKP is purely based on mathematics principles. By using ZKP technology, we are able to scale out privacy in a hardware-agnostic and more decentralized (you dont need to have a specialized hardware to run manta nodes) manner in the Web 3 world.

Web 3 Ready

Manta Network isnt building a privacy token. Instead, it is building a privacy utility. What this means is that the network is designed to enable the privatization of other cryptoassets, starting with the cryptoassets within the Polkadot and Kusama ecosystems.

Interoperability and Composability

Using Polkadot cross-consensus messaging (XCM) mechanism, Manta Network is able to communicate with other layer-1 decentralized networks within the Polkadot ecosystem. This access to native communication properties allows Manta Networks technology to provide additional benefits of privacy as a plug-and-play feature that can be enabled for other networks.

Privatize All the Things

Because our ZKP implementation privatizes on-chain transactions, Manta Networks privacy-preservation extends beyond token transactions, and can be applied for use in DeFi, metaverses, NFTs, GameFis, and other future use cases that emerge within the growing Web 3 world.

Why Polkadot

Native Interoperability & Composability

Using Polkadots XCM protocol, our layer-1 network is able to communicate, send, and receive data from other layer-1 chains that are using the Substrate framework and connected as a Polkadot parachain. This feature allows us to privatize other layer-1 cryptoassets in a secure and decentralized manner.

Performance & Security

Thanks to Polkadots Nomindated Proof of Stake (NPoS) consesus, which is fast, secure and decentralized, and WebAssembly (WASM) based runtime. Manta can bring blazingly fast private blockchain transactions.

Layer 1

By using the Substrate framework, Manta Network is able to deploy as a layer-1 network while leveraging all of the features that are available for Polkadot parachains, including XCM and security.

Project Roadmap

Q4 2020 - Project begins. Manta Network receives a Web3 Foundation Grant to build a decentralized anonymous payment system.

Q1 2021 - Manta Network raises a seed round, led by Polychain Capital. The project completes the first milestone of the W3F grant.

Q2 2021 - Manta Network publishes the updated version of its whitepaper in an e-print within the International Association for Cryptologic Research.

Q3 2021 - Manta Network secures a parachain for Calamari Network on Kusama and raises a private round led by CoinFund and ParaFi.

Q4 2021 - Dolphin Testnet Version 1 is released. Calamari Network launches community governance.

Q1 2022 - Manta Network completes its XCM design library to support cross-chain messaging within the Polkadot and Kusama ecosystems. Community collators are enabled on Calamari to decentralize the network.

Q2 2022 - Binance Labs invests in Manta Network. Dolphin Testnet Version 2 launched with significantly improved prover speed alongside additional new features.

Q3 2022 (Current) - Expanding to 30+ community collators. Launching trusted setup to transition Dolphin Testnet onto MantaPay on Calamari. Decentralized staking to launch on Calamari. Security audits for MantaPay.
